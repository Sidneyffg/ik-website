Upload it meme

Upload it meme is een namaak website van make it meme. De bedoeling is om met een groepje mensen memes te maken en dan elkaars memes raten, uplaod it meme zorgt ervoor dat je je eigen memes kan uploaden.

Chess bot

Ik begon de schaakbot in javascript te maken, toen ik een werkende versie had was hij niet zo goed. Ik herschreef hem in c++ en heb wat optimalisaties toegevoegd waardoor hij nu beter is in schaken dan ikzelf.

Leerplek

Leerplek is een website om woordjes te leren. In dit project heb ik bij verre na de meeste uren zitten, maar ik heb het nooit afgemaakt.

Craftmine

Craftmine is een Minecraft clone geschreven in c++ met openGL. Het terein genereert gebaseerd op een seed die je in kan vullen, als je dezelfde seed invult is de wereld identiek.

Strava graphs

Ik ben gaan hardlopen, dat neem ik op met Strava. Strava heeft een API, daarmee heb ik een website gemaakt die grafiekjes laat zien over je runs. Zo kan je makkelijk zien hoeveel en hoe goed je loopt.
